# FCS BFF

O FCS BFF oferece uma fachada HTTP pública para os clientes da plataforma e encaminha as chamadas para os serviços de domínio.

## Operação

- Namespace Kubernetes: `fcs-bff`
- Health check: `/health`
- Métricas: `/metrics`
- API: `/api/v1`

Consulte o [README](../README.md) para requisitos, execução local e rotas encaminhadas.
